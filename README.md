# Port_Manager

## 港口部分
1.吞吐量查询
输入：港口名字 起始时间 终止时间（按XXXX-XX-XX的格式输入）

2.不同类型货物吞吐趋势查询
输入：港口名字 起始时间 终止时间（按XXXX-XX-XX的格式输入）

3.港口货物吞吐环比
输入：港口名字 起始时间 终止时间（按XXXX-XX-XX的格式输入） 查询类型
（类型1 特定时间，一类货物占全部货物的比例）
（类型2 特定货物，一定时间段内占全年的比例）

4.不同货物流向分析
输入：货物名称

5.添加一条数据
输入：起点 终点 货物名称 数量 时间信息

## 客户部分
1.查询客户
输入：信息种类 信息
（id：身份证号 phone：手机号 address：地址）

2.增加客户信息
输入：姓名 身份证号 手机号 地址

## 登录功能
1.注册账号
输入：账号名 邮箱 密码

2.登录账号
输入：邮箱 密码

3.注销账号
输入：邮箱 密码
